A Flask API For [StreamMusic](https://github.com/gitbobobo/StreamMusic)

## 说明

此仓库为 [LrcAPI](https://github.com/HisAtri/LrcApi) 的分支，用于公开服务器<https://lrc.xms.mx>，可通过API接口<https://lrc.xms.mx/lyrics>快速使用。

由于与个人部署服务端差异较大，单独设立仓库。
